# Портфолио Python-разработчика

Современное портфолио с демонстрацией навыков: сайт автошколы, Telegram-бот магазина, парсер вакансий HH.ru, админ-панель с 2FA и многоуровневой защитой.

## Стек
- **Backend**: Python 3.11, Flask, SQLAlchemy, PostgreSQL/SQLite
- **Frontend**: HTML5, CSS3, JavaScript (ванильный)
- **Безопасность**: CSRF, ограничение запросов, заголовки безопасности, шифрование OTP, капча
- **DevOps**: Docker, docker-compose, Gunicorn, CI/CD (GitHub Actions)

## Установка и запуск
1. Клонируйте репозиторий
2. Скопируйте `.env.example` в `.env` и заполните секреты
3. Запустите через Docker:
   ```bash
   docker-compose up --build
Или локально:
   ```bash
   pip install -r requirements.txt
   flask run
```

Функционал

· Главная страница-визитка
· Контакты с email-уведомлениями
· Демо-страница автошколы с записью на занятие
· Демо-магазин с корзиной
· Парсер вакансий с кешированием и экспортом в CSV
· Админ-панель с двухфакторной аутентификацией
· REST API для получения списка проектов

Безопасность

· Пароли хешируются (werkzeug)
· CSRF-защита на всех формах
· Ограничение частоты запросов (Flask-Limiter)
· HTTP-заголовки безопасности (Talisman)
· Капча на публичных формах
· Шифрование секретов OTP
· Защита от перебора паролей
